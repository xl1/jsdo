Chrome の about:flags で Web Audio API 有効にすると使えます。
(Windows 版 14.0.814.0 dev 以上では、たぶんデフォルトで有効）

jsPlumb http://code.google.com/p/jsplumb/ を利用しています。
音楽ファイルは http://jsdo.it/assets の SVG girl theme を利用しています。
MML 変換器はとりあえずのものです http://jsdo.it/xl1/enAs

switch はダブルクリックすると 音を通す/通さない を切り替えます。
ドラッグ&ドロップでノードを動かしたりつなげたりします。
